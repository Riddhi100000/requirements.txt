## What is SafeGuard AI?
A women's safety platform with:
- 🆘 One-press SOS Alert with GPS location
- 🎤 Voice-activated SOS (say "HELP" to trigger)
- 📍 Live location sharing with trusted contacts
- 🎙️ Audio evidence recording and secure storage
- ✅ Periodic safety check-ins
